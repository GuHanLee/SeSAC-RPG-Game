# SeSAC RPG Game

A simple text-based RPG game written in C.

## Features

- **Dungeon Exploration**: Explore 10 different dungeons and battle enemies
- **Inventory System**: Collect and manage items (weapons and armor)
- **Equipment Management**: Equip weapons and armor to enhance your stats
- **Shop System**: Buy and sell items with gold
- **Combat System**: Fight enemies and earn rewards
- **Item Drops**: Random item drops from defeated enemies

## Game Mechanics

### Items
- **Weapons**: Increase attack power
- **Armor**: Increase defense power

### Shop
- Purchase better equipment with gold
- Prices vary by item grade (A급, S급, SSS급)

### Dungeons
- 10 different dungeons to conquer
- Increasing difficulty levels
- Rewards: Gold and Item drops

## How to Compile and Run

```bash
gcc -o main main.c
./main
```

## Game Controls

- Follow the on-screen prompts to navigate the game menu
- Select actions like exploring dungeons, managing inventory, visiting shop, etc.

## Requirements

- GCC compiler or any C compiler
- macOS/Linux/Windows with terminal support

## Project Structure

- `main.c`: Main game source code
- `main`: Compiled executable binary
